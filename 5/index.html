5) Faça uma aplicação web que utilize a função listImages(url), criada anteriormente, e liste os objetos recebidos na forma de cards (https://www.w3schools.com/howto/howto_css_cards.asp). Os cards devem ser dispostos em duas colunas. Note que os objetos recebidos possuem os campos de endereço para imagens, título das imagens, identificador (id) das imagens e identificador de álbum (albumId). Apresente as imagens (cujas urls são fornecidas nos objetos) no formato de thumbnail. Ao se clicar no thumbnail, a imagem em seu tamanho normal deve ser apresentada na forma de lightbox (https://www.w3schools.com/howto/howto_js_lightbox.asp). Utilize como base para seu código o material disponível em https://www.w3schools.com/howto/default.asp. (1 ponto)
